El propósito de este codelab no es convertirse en un tutorial de Git, como mucha gente lo dice ‘no es necesario reinventar la rueda' puesto ya que existe infinidad de tutoriales muy detallados tanto avanzados como básicos.

En este codelab quisiera mencionar algunos sitios a los cuales acudir cuando necesite ayuda o aprender una nueva funcionalidad.

Pagina oficial de Git

Claramente podemos buscar todo de git en su pagina oficial. podemos encontrar libros, documentación, entre otros. La pagina esta en multiples idiomas y el libro explica funcionalidades de Git de una manera muy clara, si nos gusta leer prácticamente podemos convertirnos en expertos en la pagina official.

Git básico

Si solo vamos a tener que trabajar funcionalidades básicas, podemos visitar la pagina de git - la guía sencilla de Roger Dudler, en la cuál, como lo explica la misma pagina es una guía sencilla para comenzar con git

Documentación offline

Si tenemos instalado git podemos obtener ayuda offline, esto lo menciona en la introducción del libro de Git - obteniendo ayuda.

Git help <verb>
Git <verb> --help

Miremos algunos ejemplos

Git help config

La documentación sera lanzada en el navegador pero esta trabaja de forma offline.
